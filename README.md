# SAQMS — Smart Appointment & Queue Management System

A web-based clinic appointment and queue management system built for DSC-11 Software Engineering.

## Tech Stack

- **Frontend:** React
- **Backend:** Node.js / Express
- **Database:** PostgreSQL
- **Auth:** JWT + bcrypt
- **Notifications:** SendGrid (email), Twilio (SMS)

## Prerequisites

- Node.js v18+
- PostgreSQL 14+
- Redis
